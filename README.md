# Цаг агаарын мэдээний карт үүсгэгч

Дархан-Уул аймгийн УЦУОШТ-ийн Instagram (1080×1080) цаг агаарын мэдээний карт үүсгэдэг статик веб апп.

## Файлын бүтэц

```
weather-card/
├── index.html        ← Үндсэн апп (нээхэд шууд ажиллана)
├── assets/           ← Зургууд (index.html-ээс салгасан)
│   ├── logo.png
│   ├── darkhan.jpg
│   ├── orkhon.jpg
│   ├── khongor.jpg
│   └── sharyngol.jpg
└── README.md
```

> Өмнө нь зургууд бүгд `index.html` дотор base64 хэлбэрээр шигдсэн байсан (≈577 KB).
> Одоо салгаснаар `index.html` ердөө **34 KB** болж, хуудас хурдан ачаалагддаг болсон.

## GitHub Pages дээр байрлуулах — шат дараалал

### 1. GitHub бүртгэл ба repository
1. https://github.com дээр бүртгэлээ нээнэ (байхгүй бол).
2. Баруун дээд булангийн **+ → New repository** дарна.
3. **Repository name**: жишээ нь `weather-card`.
4. **Public** сонгоно (GitHub Pages үнэгүй хувилбарт public байх шаардлагатай).
5. **Create repository** дарна.

### 2. Файлуудаа байршуулах
1. Шинэ repository хуудсан дээр **Add file → Upload files** дарна.
2. Энэ хавтасны **бүх агуулгыг** (index.html, assets хавтас бүхэлдээ, README.md) чирж оруулна.
   - `assets` хавтсыг доторх зурагтай нь хамт чирж болно; GitHub бүтцийг нь хадгална.
3. Доор нь **Commit changes** дарна.

### 3. Pages-ийг идэвхжүүлэх
1. Repository → **Settings** (дээд цэс).
2. Зүүн талын жагсаалтаас **Pages**.
3. **Source** хэсэгт **Deploy from a branch** сонгоно.
4. **Branch**-д `main` (эсвэл `master`), хавтсанд `/ (root)` сонгоод **Save**.
5. 1–3 минут хүлээнэ.

### 4. Хаягаа нээх
Дараах хаягаар нээгдэнэ:

```
https://<таны-хэрэглэгчийн-нэр>.github.io/weather-card/
```

Хэрэв `index.html` root-д байвал энэ хаяг шууд аппыг харуулна.

## Санамж
- Апп нь Google Fonts болон html2canvas сангуудыг интернэтээс (CDN) татдаг тул онлайн байх үед л зөв ажиллана.
- Зураг солих бол `assets` доторх файлыг ижил нэрээр солиод дахин upload хийхэд болно.
- Хэрэв суурин нэмэх бол `index.html` доторх `SOUMS` массивт мөр нэмж, `img:'assets/шинэ.jpg'` гэж заана.
